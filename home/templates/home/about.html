About JaCros Design

Currently JaCros design is a one man band headed entirely by Jack Crosbie
Jack recently finished a level 6 Diploma in Software Development from the Code Institute of Dublin, graduating in the summer of 2022.
Prior to this Jack has worked in many various industries and positions but his background is mainly in the music industry
He holds a degree in music from the Cork School of Music in county Cork, Ireland.
After the pandemic hit Jack decided it was time to change careers and this lead him to the Code Institute of Dublin.
From his time here he learnt about HTML, CSS, JavaScript, Python and using various packages such as Django, Bootstrap and PostgreSQL.
Below is examples of projects he completed and examples of the different languages learnt.

Project One

The first project I undertook for my course was to build a website. I decided to use a local band and built them a website based around information they provided for me. The project was built using HTML and CSS only. Features of the website included bios of each band member, sample setlists and also a live video of the band playing.

Project Two

The second project I completed as part of my course was a JavaScript project. I chose to do an interactive quiz which selected questions from a selected database. The quiz randomly selected questions from a set and was therefor replayable. It included a score tracker and graphics to show users want answer they were selecting and the result.

Project Three

The third project undertaken was a Python Essentials project which involed us building a database to access sets of information. As a football fan I opted to build a database with 4 different football teams and their previous years stats. You could then organise the data to display top scorers, oldest players and other various stats. The database was hosted on Heroku after it's completion.

Project Four

The fourth project of the course involved building a website with a backend data base using django. I opted to make a restaurant website with a reservations system, menu, contact form and other features. I implemented full CRUD functionality with the reservations app and also utilised django-AllAuth so users could register and login and out.
